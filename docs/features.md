# Features

## Core Calculator Operations
- **Addition** – add two or more numbers
- **Subtraction** – subtract numbers
- **Multiplication** – multiply numbers
- **Division** – divide numbers with divide-by-zero error handling

## Display
- Show current input and running expression
- Show result after pressing equals
- Support decimal point input

## Controls
- **Clear (C)** – reset calculator to default state
- **Backspace (⌫)** – delete last entered character
- **Equals (=)** – evaluate and display result
- **Negative toggle (+/-)** – toggle positive/negative on current number
- **Percentage (%)** – convert current number to percentage

## UX & Design
- Responsive layout that works on mobile and desktop
- Keyboard input support (numbers, operators, Enter, Backspace, Escape)
- Tailwind CSS styled with a modern dark theme
- Error state display (e.g. "Error" for invalid operations)
